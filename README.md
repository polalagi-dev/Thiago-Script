# Instructions
To start you need: **git**

## Install git
Switch to root mode by using: ```su```

Install git By Using: ```apt install git```

If this doesn't work try this: ```apt-get install git```


## Setup Thiago

First, Clone Repository Using: ```git clone https://github.com/polalagi-dev/Thiago-Script```

Then, Change Directory To Thiago-Script Using: ```cd Thiago-Script```

Then, Give Permission To install.sh Using: ```chmod +x install.sh```

Finally, Run The Script: ```./install.sh```
